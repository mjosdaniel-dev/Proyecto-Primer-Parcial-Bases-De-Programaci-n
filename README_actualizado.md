# 🌦️ Automatización del Análisis y Visualización de Datos Climáticos

Proyecto desarrollado para la asignatura **Bases de Programación**, cuyo
objetivo es automatizar el proceso de carga, limpieza, análisis,
manipulación y visualización de datos climáticos utilizando Python.

## 📌 Descripción

Este proyecto implementa un flujo completo de análisis de datos mediante
herramientas de ciencia de datos en Python. A partir de un conjunto de
datos climáticos, se realizan procesos de limpieza, análisis
exploratorio (EDA), generación de estadísticas descriptivas,
manipulación de datos y creación de visualizaciones que permiten
comprender el comportamiento de variables como temperatura, humedad,
precipitación y velocidad del viento.

## 🎯 Objetivo

Desarrollar un programa en Python que permita:

-   Cargar datos climáticos.
-   Limpiar y preparar los datos para el análisis.
-   Obtener estadísticas descriptivas.
-   Realizar análisis exploratorio de datos (EDA).
-   Manipular la información mediante filtros y agrupaciones.
-   Generar visualizaciones automáticas.
-   Exportar los resultados obtenidos.

## 🛠️ Tecnologías utilizadas

-   Python 3
-   Jupyter Notebook
-   Pandas
-   NumPy
-   Matplotlib
-   OpenPyXL

## 📊 Funcionalidades

-   Importación y carga de datos.
-   Limpieza de valores faltantes y duplicados.
-   Resumen estadístico del conjunto de datos.
-   Análisis exploratorio (EDA).
-   Temperatura promedio por ciudad.
-   Precipitación promedio por ciudad.
-   Humedad promedio por ciudad.
-   Velocidad promedio del viento.
-   Temperatura promedio por mes.
-   Creación de nuevas columnas.
-   Exportación de resultados a Excel.
-   Generación automática de gráficos.

## 📈 Visualizaciones

El proyecto genera diferentes gráficos para facilitar la interpretación
de los datos, entre ellos:

-   Gráficos de barras.
-   Histogramas.
-   Diagramas de caja (Boxplot).
-   Gráficos de líneas.
-   Gráficos automáticos por ciudad.
